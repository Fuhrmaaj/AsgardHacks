What is Asgard Hacks Anyways?

Asgard Hacks is an opportunity for people to participate in Hacktoberfest.

Hacktoberfest is a competition put on every year by GitHub inviting people to make 3 code contributions to any public project. If, before the end of October, a person has made the 3 contributions (and signed up to the event beforehand) GitHub will send you a Hacktoberfest t-shirt and a sticker pack for free!

Code Contributions Seem Hard

THe point of Asgard Hacks is to provide a comfortable entry point for everyone. There will be a set up project to contribute to with "quests" of varying difficulty. The first session will also be an intro session where we make sure everyone is signed up to the event and a workshop where we teach people about git and how to contribute.

Experienced mentors will be there to help anyone struggling or who has questions. The goal is to learn a bit about coding and to help people complete the event successfully.

When Is It?

Every Tuesday evening there will be coding sessions in the Asgard Green community Centre.

What Do I Need?

You will need to bring a computer. It doesn't matter how old or portable it is. Feel free to bring a desktop if that's all you have, we'll get you set up.
